GitHub 一键发布设置
填好下面这些后，"🚀 一键发布"按钮会出现在右上角。点它，网页直接把你的草稿写到 GitHub 仓库（图片 + data.json 一起 commit），不用再手动下载 zip。
去 github.com/settings/personal-access-tokens → Generate new token (fine-grained)：
• Repository access：Only selected repositories，选你这一个仓库
• Repository permissions → Contents: Read and write
• Expiration：建议 90 天，到期重新生成
⚠ Token 只存在你当前这个浏览器里。换电脑要重新填。如果电脑会被别人用，别填这里——继续用导出 zip 的方式更安全。
